# Mimoun.co — Landing page

Single-file landing page for [mimoun.co](https://mimoun.co).

## Structure

```
index.html          — full page, no framework, no build step
assets/
  mimoun-logo.png   — logo (used at height 28px in header)
robots.txt
sitemap.xml
```

## Running locally

Double-click `index.html` — it opens directly in any browser.

## Deploying

Drop the repo root on [Vercel](https://vercel.com) or any static host. No build command needed.
